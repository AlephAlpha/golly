JvN is short for John von Neumann, a pioneer in the field of cellular automata. The JvN algorithm supports 3 rules:

JvN29
The original 29-state CA as created by von Neumann. See [1] and [2].

Nobili32
An extension of JvN29 created by Renato Nobili. This rule adds three new states to allow signal crossing.

Hutton32
Created by Tim Hutton, this rule uses the same states as Nobili32 but has a modified rule table.
