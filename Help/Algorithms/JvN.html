JvN is short for John von Neumann.

The JvN algorithm supports 3 rules:

  • JvN29 - the original 29-state CA as created by von Neumann. See: [1][2]
  • Nobili32 - an extension of JvN29, adding three new states to allow signal crossing.
  • Hutton32 - uses the same states as Nobili32 but has a modified rule table.
