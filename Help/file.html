NOTE: [key] next to a menu item indicates a keyboard shortcut.

New Pattern

Creates a new, empty universe and switches to a large scale ready for editing cells. Use the Preferences dialog to set the scale (initially 1:16), the cursor mode, and whether any selection should be removed.

Open Pattern...

Brings up the standard file browser so you can select a pattern file. Golly can read all the common pattern formats: RLE, PC Life 1.05/1.06, and text patterns like "ooo...ooo". It can also read patterns in a "macrocell" format which allows huge, highly repetitive patterns to be stored in a very compact way. Note that Golly automatically turns on the Use Hashing option when it reads a macrocell file. Such files usually have a .mc extension.

All the above formats are text based and can have DOS/Mac/Unix line endings, or they can be compressed in a .gz file.

Use the Preferences dialog to set the cursor mode, and whether any selection should be removed.

Open Clipboard

Opens a pattern stored in the clipboard. The References section has links to a number of good sources for clipboard patterns. [There is currently no global clipboard support in the X11 version; you can only open a pattern that has been saved by the Edit menu's Cut/Copy commands.]

Open Recent

This submenu lets you open a recently loaded pattern file. The most recent file is always at the top. The maximum number of files remembered is determined by a setting (initially 20) which you can change in the Preferences dialog.

Save Pattern...

Brings up the standard file saving dialog so you can save the current pattern in RLE format (but only if all live cells are within coordinates of +/- 1 billion). If the hashing option is on then you can also save the pattern in macrocell format.

Preferences...  [,]

Opens the Preferences dialog so you can change various settings. [On the Mac, this item is in the application menu.]
