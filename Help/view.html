NOTE: [key] next to a menu item indicates a keyboard shortcut.

Fit Pattern  [f]

Changes the scale and location so the pattern fits snugly within the current view.

Middle  [m]

Moves the cell at 0,0 to the middle of the view.

Full Screen  [F1 on Mac, F11 on other platforms]

Switches to full screen mode, automatically hiding the menu bar, the status bar, the tool bar and the scroll bars. To exit from this mode just hit the same function key or any unused key, like return. [Currently not supported on X11.]

Zoom In  [* or ]]

Zooms in by doubling the current scale. If the scale is 1:16 then you cannot zoom in any further.

Zoom Out  [/ or []

Zooms out by halving the current scale. Golly has an unbounded universe so you can zoom out as far as you like!

Show Status Bar  [;]

Shows or hides the status bar at the top of the window.

Show Tool Bar  [']

Shows or hides the tool bar at the left edge of the window. [On X11 the tool bar is above the status bar.]

Show Grid Lines  [l]

Shows or hides the grid lines (only drawn at scales 1:8 or 1:16).

Black on White  [b]

Toggles the colors used for cell states. If ticked then live cells are black and dead cells are white, otherwise vice versa.

Buffered

If ticked then window buffering is used to avoid flicker, but it does slow down pattern rendering a bit so you might prefer to live with the flicker. Note that buffering is always used if grid lines are displayed or if a selection is visible.

This option is disabled on Mac OS X because windows are automatically buffered.

Pattern Info  [i]

Displays any comments in the current pattern file.
